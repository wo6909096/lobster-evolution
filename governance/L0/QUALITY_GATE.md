# QUALITY_GATE

任何能力/流程升级前必须满足：

1. 可复现：按文档可重复执行。
2. 可验证：有可观测指标（成功率/耗时/步骤/调用成本）。
3. 可解释：输入、输出、失败边界明确。
4. 可回滚：有上一稳定版本与回滚条件。

未满足任一项：禁止升级到 L1。
